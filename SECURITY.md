# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the maintainers
rather than opening a public issue.

While this repository is private, contact the maintainers directly via internal
channels. Once the project is open-sourced, this section will be updated with a
public reporting address.

Please include:

- Description of the issue
- Steps to reproduce
- Affected versions
- Potential impact

We aim to acknowledge reports within 5 business days.
